# **App Name**: GuardianLife

## Core Features:

- User Registration & Authentication: Secure user registration and authentication via email/phone + OTP, including profile editing.
- Health Tracking Module: Track heart rate, steps, calories burned, and sleep duration, with daily/weekly/monthly reports and data visualization.
- Geo-Fencing System: Define safe zones and trigger alerts when the user unexpectedly exits a geo-fenced area, with background location tracking optimized for low battery consumption.
- Emergency Alert Button (SOS): One-tap SOS button that sends real-time location, user details, and an alert message to emergency contacts and caregivers; includes option to auto-call emergency contact.
- AI Health Chatbot: AI chatbot providing basic, non-diagnostic health advice, medication reminders, stress management tips, and emergency guidance using user data for personalized responses (privacy-safe). This AI will use a tool to remember important details of the user health state.
- Notifications & Alerts: Push notifications for geo-fence breaches, abnormal health values, medication reminders, and SOS confirmation.
- Admin / Caregiver Dashboard: Dashboard for viewing registered users, monitoring live location (with permission), receiving emergency alerts, and viewing health summaries.

## Style Guidelines:

- Primary color: HSL (120, 60%, 40%) - A vibrant, energetic green (#4CAF50), signifying health and well-being.
- Background color: A soft, desaturated green (HSL 120, 20%, 95%) - a pale off-white (#F0FAF2), creating a calming backdrop.
- Accent color: HSL (30, 70%, 45%) - A light-brown (#B37A0D) for key interactive elements and highlights.
- Body font: 'PT Sans', a humanist sans-serif.
- Headline font: 'Belleza', a humanist sans-serif appropriate to art, design and fashion.
- Use clear, modern icons for health metrics and navigation.
- Subtle animations to enhance user interaction and provide feedback.
